# Katunog Audio Bundle

This folder intentionally contains only the files used by the simulator:

- `audio_manifest.csv`: 70 rows for Katunog control number `PIISD02596`
- `audio/PIISD02596__gabbang/`: 68 playable Gabbang strikes and 2 reference recordings

The source archive downloaded during research contained many more instruments.
Those files are excluded from this repository.

Audio and metadata source: https://katunog.asti.dost.gov.ph

Terms of use: https://katunog.asti.dost.gov.ph/client/terms
